緊急車両の出動情報をWebサイトの形で掲載する

## データソース
データは[emergency-dispatch](https://github.com/oageo/emergency-dispatch)からJSON形式で取得しています。このサイトにおいては、emergency-dispatchで得られたJSONファイルを人間に見やすい形で表示することを行っています。
